# xray-script

Xray 一键安装脚本，基于网络跳跃原版魔改，支持IPv4/IPv6 VPS，支持与宝塔面板共存

```shell
wget -N --no-check-certificate https://raw.githubusercontent.com/Misaka-blog/xray-script/main/xray.sh && bash xray.sh
```
